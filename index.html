WoRDLE GaME

How to Play:

Welcome to the WoRDLE GaME!

  • *INSTRUCTIONS*
  • This is a Two-Player game where each player takes turns guessing a FRUIT that has 5-letters.
  • Player 1 starts the game and guesses by clicking letters on the keyboard.
  • Once Player 1 enters a word, Player 2 takes their turn.
  • HINT: The game will show which letters are correct by highlighting them in green.
  • The first player to guess the word correctly wins!
  • By click the "Reset" button to restart the game with a new word.
  • If you're stuck, use the "Hint" button (but only once per game).
